## 📖 第1課（だい1か）フットサルって、何（なん）でしたっけ？
 
**トピック:** 好（す）きなもの好（す）きなこと (สิ่งที่ชอบ)
 
**เป้าหมาย (A2):** ฟังเพื่อนพูดง่ายๆ เกี่ยวกับกีฬาที่ชอบ และเข้าใจเหตุผล/จุดเด่นของกีฬานั้น
 
**โครงเรื่อง:** สนทนาเรื่องกีฬาที่ชอบ (①休（やす）みの日（ひ）はスポーツをしています) + การชวนไปงานอีเวนต์ (②イベントに誘（さそ）いましょう)

### 📝 ไวยากรณ์ (文法（ぶんぽう）ノート)
 
#### ➊ N ＋ でしたっけ？
> フットサルって、何（なん）でしたっけ？ (ฟุตซอลคืออะไรนะ...ทวนหน่อย)
 
- ใช้เมื่ออยาก**ยืนยัน/ทวนความจำ**กับอีกฝ่าย
- ต่างจาก 何（なん）ですか？ (ไม่รู้จักเลย ถามความหมาย) — 何（なん）でしたっけ？ = เคยรู้มาก่อนแต่จำเลือนๆ
- มักใช้คู่กับ「って」หยิบยกหัวข้อขึ้นมาก่อน เช่น フットサルって〜
- ต่อกับประโยคได้: **S（普通形（ふつうけい））んでしたっけ？**
- ตัวอย่าง:
  - サッカーって、1（いっ）チーム9人（にん）でしたっけ？ → いいえ、11人（にん）ですよ。
  - フットサルの試合（しあい）って、来月（らいげつ）の土曜日（どようび）でしたっけ？ → え、日曜日（にちようび）ですよ。
#### ➋ V（動詞（どうし）辞書形（じしょけい））＋ることになった
> 私（わたし）の友（とも）だちが、バリ舞踊（ぶよう）を踊（おど）ることになったんです。
 
- บอก**แผนที่ถูกกำหนดแล้ว** โดยไม่ใช่การตัดสินใจเอง แต่เกิดจาก**สถานการณ์/การพูดคุยตกลงกัน**
- ต่อกับ ナイ形（けい） ได้ → หมายถึง "ตกลงว่าจะไม่ทำ"
- ตัวอย่าง:
  - 友（とも）だちに誘（さそ）われて、週末（しゅうまつ）、宝塚（たからづか）を見（み）に行（い）くことになりました。
  - 予算不足（よさんぶそく）のため、今年（ことし）のフェスティバルは、開催（かいさい）されないことになった。
#### ➌ V（て形（けい））＋くれませんか？
> もしよかったら、見（み）に来（き）てくれませんか？ (ถ้าสะดวก มาดูได้ไหมครับ)
 
- ใช้**ขอร้อง**สิ่งที่ผู้พูดอยากให้เกิดขึ้น (เป็นประโยชน์กับตนเอง)
- ในบทนี้ใช้ชวนคนมาดูอีเวนต์ที่ตัวเองร่วมแสดง
- แบบลำลอง (เพื่อนสนิท): V（て形（けい））＋くれない？ เช่น 見（み）に来（き）てくれない？
- ตัวอย่าง:
  - 友（とも）だちが行（い）けなくなってしまったので、落語（らくご）、いっしょに行（い）ってくれませんか？
  - 日曜日（にちようび）の野球（やきゅう）の試合（しあい）、出（で）てくれない？

### 🎭 ความรู้เสริมวัฒนธรรม
 
**落語（らくご） — ราคุโกะ**
ศิลปะการเล่าเรื่องตลกดั้งเดิมของญี่ปุ่น เริ่มในยุคเอโดะ ผู้เล่าคนเดียว (落語家（らくごか）) แสดงเป็นหลายตัวละครโดยใช้แค่พัด（扇子（せんす））กับผ้าเช็ดมือ（手（て）ぬぐい）เป็นอุปกรณ์ จบด้วยมุกตลก (オチ)
 
**にほん vs にっぽん**
- にほん = ใช้ในชีวิตประจำวันทั่วไป, คำประสม เช่น 日本語（にほんご）, 日本料理（にほんりょうり）
- にっぽん = อ่านแบบเก่ากว่า มักใช้ในบริบททางการ/ดั้งเดิม

### 📝 ไวยากรณ์ (文法（ぶんぽう）ノート)
 
#### ➊ S（普通形（ふつうけい））＋でしょう？
> おもしろかったでしょう？ (สนุกใช่ไหมล่ะ)
 
- ใช้**ขอความเห็นด้วย/ยืนยัน**สิ่งที่ผู้พูดเชื่อว่าเป็นจริง โดยคาดว่าอีกฝ่ายจะรู้สึกเหมือนกัน
- ออกเสียงลงท้ายแบบ**เสียงสูงขึ้น** (rising intonation)
- คำนาม/นะ-adjective → **Nでしょう？ / ナA-でしょう？**
- ใช้กับคนสนิท (เพื่อน/ครอบครัว) เท่านั้น — ใช้กับผู้ใหญ่กว่า/เจ้านายจะดูเสียมารยาท
- ตัวอย่าง:
  - この曲（きょく）、いい曲（きょく）でしょう？ → はい。だれの曲（きょく）ですか？
  - おすすめしてもらったドラマ、見（み）ました。→ どうだった？よかったでしょう？
#### ➋ S（普通形（ふつうけい））＋気（き）がする
> ストーリーがちょっと短（みじか）かった気（き）がしました。 (รู้สึกว่าเรื่องมันสั้นไปนิดนึง)
 
- ใช้บอก**ความประทับใจ/ความเห็นแบบนุ่มนวล** ไม่พูดตรงๆ
- ในบทนี้ใช้เลี่ยงการพูดความเห็นเชิงลบแบบตรงๆ
- นะ-adjective → **ナA-な気（き）がする**
- ตัวอย่าง:
  - でも、最後（さいご）のシーンは、ちょっと、いまいちな気（き）がしました。
  - うーん、おもしろかったけど、ストーリーがちょっと難（むずか）しかった気（き）がする。
#### ➌ S（普通形（ふつうけい））＋って N
> 歴史（れきし）を変（か）えようとする敵（てき）と戦（たたか）うってゲームです。 (เป็นเกมที่ต้องสู้กับศัตรูที่พยายามเปลี่ยนประวัติศาสตร์)
 
- ใช้**อธิบายเนื้อหาอย่างเจาะจง** เช่น อธิบายเกม/ละคร
- ต่อกับรูปธรรมดา (普通形（ふつうけい）) คำนาม/นะ-adjective → **N1だってN2 / ナA-だってN**
- 「って」= ภาษาพูดลำลอง ถ้าเป็นภาษาเขียน/ทางการใช้ **という** แทน
- ตัวอย่าง:
  - 最後（さいご）は、お金（かね）じゃなくて、人（ひと）とのつながりが大事（だいじ）だって話（はなし）です。
  - 主人公（しゅじんこう）の男（おとこ）の子（こ）が、仲間（なかま）といっしょに次々（つぎつぎ）に事件（じけん）を解決（かいけつ）していくって映画（えいが）です。
#### ➍ สำนวนเชื่อมประโยค（接続（せつぞく）の表現（ひょうげん））
> それに、ゲームだけじゃなくて、舞台（ぶたい）とかミュージカルとかアニメとかもあって、いろいろ楽（たの）しめるんです。
 
ใช้เชื่อมประโยคเวลาต้อง**เรียงเหตุผล/จุดเด่นหลายๆ อย่าง** ต่อกัน แบ่งตามความสัมพันธ์ของประโยค:
 
| กลุ่มความหมาย | คำเชื่อม | ตัวอย่าง |
|---|---|---|
| บอกผล/สรุปจากเหตุผลก่อนหน้า | だから (เพราะฉะนั้น) | このレストラン、この前（まえ）『孤独（こどく）のグルメ』に出（で）てきたんですよ。だから、こんなに混（こ）んでいるんです。 |
| บอกผล (ภาษาพูด) | それで／で (ดังนั้น) | ライブのチケットがとれたんです。それで、その日（ひ）は休（やす）みをとりました。 |
| ขัดแย้งกับประโยคก่อนหน้า | でも (แต่) | 趣味（しゅみ）はオンラインゲームです。でも、スポーツもよくします。 |
| เพิ่มเติมสิ่งอื่น | あと／それと／それに (นอกจากนี้) | この俳優（はいゆう）さん、歌（うた）も上手（じょうず）なんです。あと、ダンスもできるんですよ。 |

## 📖 第3課（だい3か）引（ひ）っ越（こ）しの準備（じゅんび）はどうですか？
 
**トピック:** どこに住（す）む？ (จะอาศัยอยู่ที่ไหน?)
 
**เป้าหมาย (A2):** ดูข้อมูลอสังหาริมทรัพย์จากเว็บไซต์ แล้วหาข้อมูลที่จำเป็น เช่น ค่าเช่า ขนาดห้อง ทำเลที่ตั้งได้
 
**โครงเรื่อง:** อ่านประกาศห้องเช่า (物件情報（ぶっけんじょうほう）) + ฟังบทสนทนาเปรียบเทียบย่านต่างๆ ที่จะย้ายไปอยู่ + ขอความช่วยเหลือเรื่องการย้ายบ้าน

### 📝 ไวยากรณ์ (文法（ぶんぽう）ノート)
 
#### ➊ S（普通形（ふつうけい））＋なら、～
> 住（す）むなら、どのあたりがおすすめですか？ (ถ้าจะอยู่ แถวไหนดี?)
> 家賃（やちん）が安（やす）いところがいいなら、芝浜（しばはま）はどう？
 
- แสดง**เงื่อนไขตั้งต้น**ก่อนถาม/แนะนำ ในบทนี้ใช้พูดเงื่อนไขก่อน แล้วค่อยถามคำถามหรือให้คำแนะนำ
- ต่างจาก Nなら (ที่เรียนใน初級2 บท4 ใช้ตอบคำถามโดยอ้างอิงคำพูดของอีกฝ่าย) — ในบทนี้「なら」ต่อกับ**ประโยค**ได้ด้วย ไม่ใช่แค่คำนาม ต่อกับรูปธรรมดา (普通形（ふつうけい）) นะ-adjective → **ナA-なら**
- นอกจากถาม/แนะนำ ยังใช้ตอน**เสนอตัวช่วยเหลือ**หรือ**ให้คำแนะนำ**ได้ด้วย
- ตัวอย่าง:
  - 部屋（へや）を借（か）りるんですけど、1人（ひとり）でだいじょうぶでしょうか……。→ 心配（しんぱい）なら、不動産屋（ふどうさんや）にいっしょに行（い）きましょうか？
  - 1人（ひとり）で住（す）むなら、ワンルームで十分（じゅうぶん）ですよ。
#### ➋ N ＋っぽい
> 下町（したまち）っぽい雰囲気（ふんいき）ですね。 (มีบรรยากาศแบบชุมชนเก่าๆ)
 
- แสดงความหมาย**"ให้ความรู้สึกเหมือน N" หรือ "คล้ายกับ N"**
- ในภาษาพูดลำลอง ยังต่อกับประโยคเพื่อแสดงการคาดเดา "ดูเหมือนว่าจะ..." ได้ด้วย เช่น あそこにコンビニができるっぽいですよ (ดูเหมือนแถวนั้นจะมีร้านสะดวกซื้อเปิดนะ)
- มีคำที่ติดปากกลายเป็นคำศัพท์ไปแล้ว ต่อกับคำหลายชนิด ไม่ใช่แค่คำนาม เช่น 油（あぶら）っぽい (มันๆ), 子（こ）どもっぽい (เด็กๆ), 安（やす）っぽい (ดูถูกๆ), 怒（おこ）りっぽい (ขี้โมโห), 忘（わす）れっぽい (ขี้ลืม)
- ตัวอย่าง:
  - このマンションの入（い）り口（ぐち）は、ホテルっぽくておしゃれだ。
  - あの黒（くろ）っぽい屋根（やね）の家（いえ）が、田中（たなか）さんの家（いえ）ですよ。
#### ➌ V（て形（けい））＋もらえたら、～
> 引（ひ）っ越（こ）しを手伝（てつだ）ってもらえたら、とてもありがたいんですが。 (ถ้าช่วยย้ายบ้านให้หน่อยจะขอบคุณมากเลย)
 
- ใช้**บอกสิ่งที่อยากให้คนอื่นทำให้อย่างสุภาพ** ในบทนี้ใช้ตอนขอความช่วยเหลือเรื่องย้ายบ้าน
- มาจาก V-てもらう รูปความสามารถ (可能形（かのうけい）) คือ V-てもらえる + たら
- หลัง V-てもらえたら มักตามด้วยคำแสดงความรู้สึกของผู้พูด เช่น ありがたいです (ขอบคุณมาก), 助（たす）かります (ช่วยได้มากเลย), うれしいです (ดีใจมาก)
- จบประโยคด้วย んですが ได้เช่นกัน
- ตัวอย่าง:
  - 荷物（にもつ）を運（はこ）ぶのを手伝（てつだ）ってもらえたら、とても助（たす）かります。
  - この住所（じゅうしょ）に荷物（にもつ）を送（おく）ってもらえたら、助（たす）かるんですが。
  - このあたりの物件情報（ぶっけんじょうほう）、いろいろと教（おし）えてもらえたら、うれしいです。

### 🏠 ความรู้เสริม — การเช่าห้องพักในญี่ปุ่น
 
**不動産会社（ふどうさんがいしゃ）と住宅情報（じゅうたくじょうほう）サイト**
คนหาห้องเช่าผ่านบริษัทนายหน้าอสังหาริมทรัพย์ (ตามใกล้สถานีรถไฟ) หรือเว็บไซต์ข้อมูลที่อยู่อาศัยอย่าง SUUMO, HOME'S
 
**หน่วยวัดเฉพาะของญี่ปุ่น**
- **畳（じょう）／坪（つぼ）** — หน่วยวัดพื้นที่: 1 จัง (畳) = ขนาดเสื่อทาทามิ 1 ผืน ≈ 1.62 ตร.ม. (ห้อง 6 จังถือว่ามาตรฐาน, 4.5 จัง = ห้องเล็ก); 1 สึโบะ (坪) ≈ 3.3 ตร.ม. ใช้วัดที่ดิน/อาคารขนาดใหญ่
- **合（ごう）／升（しょう）** — หน่วยวัดของเหลว/ธัญพืช: 1 โกะ = 180 มล. ใช้วัดปริมาณข้าวสารตอนหุงหรือปริมาณเหล้าสาเก
**ユニットバス** — ห้องน้ำรวม (อ่างอาบน้ำ/ฝักบัว + โถส้วม + อ่างล้างหน้าในห้องเดียว) แบบเดียวกับโรงแรม ห้องที่มีแบบนี้ค่าเช่าจะถูกกว่า
 
**プロパンガス vs 都市（とし）ガス** — ก๊าซที่ใช้ในบ้าน มี 2 แบบ: ก๊าซในเมือง (ถูกกว่า, พบมากในเมืองใหญ่) กับก๊าซโพรเพน/แก๊สถัง (พบมากในชนบท, ราคาแพงกว่า)
 
**外国人（がいこくじん）の入居（にゅうきょ）** — เจ้าของบ้านบางรายยังลังเลให้ชาวต่างชาติเช่าเพราะกังวลเรื่องภาษา/วัฒนธรรม แต่ปัจจุบันมีตัวเลือกห้องเช่าสำหรับชาวต่างชาติมากขึ้น และมีบริษัทนายหน้าที่ให้บริการภาษาต่างประเทศเพิ่มขึ้นเรื่อยๆ
