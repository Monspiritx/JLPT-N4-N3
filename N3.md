📖 第1課（だい1か）フットサルって、何（なん）でしたっけ？

トピック: 好（す）きなもの好（す）きなこと (สิ่งที่ชอบ)

เป้าหมาย (A2): ฟังเพื่อนพูดง่ายๆ เกี่ยวกับกีฬาที่ชอบ และเข้าใจเหตุผล/จุดเด่นของกีฬานั้น

โครงเรื่อง: สนทนาเรื่องกีฬาที่ชอบ (①休（やす）みの日（ひ）はスポーツをしています) + การชวนไปงานอีเวนต์ (②イベントに誘（さそ）いましょう)


🔤 คันจิของบท（漢字（かんじ）のことば）

คันจิ＋ふりがなความหมาย (ไทย)野球（やきゅう）เบสบอล弱（よわ）いอ่อนแอ応援（おうえん）เชียร์, ให้กำลังใจ伝統的（でんとうてき）（な）แบบดั้งเดิม, แบบประเพณี約束（やくそく）สัญญา, นัดหมาย負（ま）けるแพ้勝（か）つชนะ誘（さそ）うชักชวน得意（とくい）（な）ถนัด, เก่ง座（すわ）るนั่ง


📝 ไวยากรณ์ (文法（ぶんぽう）ノート)

➊ N ＋ でしたっけ？


フットサルって、何（なん）でしたっけ？ (ฟุตซอลคืออะไรนะ...ทวนหน่อย)




ใช้เมื่ออยากยืนยัน/ทวนความจำกับอีกฝ่าย
ต่างจาก 何（なん）ですか？ (ไม่รู้จักเลย ถามความหมาย) — 何（なん）でしたっけ？ = เคยรู้มาก่อนแต่จำเลือนๆ
มักใช้คู่กับ「って」หยิบยกหัวข้อขึ้นมาก่อน เช่น フットサルって〜
ต่อกับประโยคได้: S（普通形（ふつうけい））んでしたっけ？
ตัวอย่าง:

サッカーって、1（いっ）チーム9人（にん）でしたっけ？ → いいえ、11人（にん）ですよ。
フットサルの試合（しあい）って、来月（らいげつ）の土曜日（どようび）でしたっけ？ → え、日曜日（にちようび）ですよ。





➋ V（動詞（どうし）辞書形（じしょけい））＋ることになった


私（わたし）の友（とも）だちが、バリ舞踊（ぶよう）を踊（おど）ることになったんです。




บอกแผนที่ถูกกำหนดแล้ว โดยไม่ใช่การตัดสินใจเอง แต่เกิดจากสถานการณ์/การพูดคุยตกลงกัน
ต่อกับ ナイ形（けい） ได้ → หมายถึง "ตกลงว่าจะไม่ทำ"
ตัวอย่าง:

友（とも）だちに誘（さそ）われて、週末（しゅうまつ）、宝塚（たからづか）を見（み）に行（い）くことになりました。
予算不足（よさんぶそく）のため、今年（ことし）のフェスティバルは、開催（かいさい）されないことになった。





➌ V（て形（けい））＋くれませんか？


もしよかったら、見（み）に来（き）てくれませんか？ (ถ้าสะดวก มาดูได้ไหมครับ)




ใช้ขอร้องสิ่งที่ผู้พูดอยากให้เกิดขึ้น (เป็นประโยชน์กับตนเอง)
ในบทนี้ใช้ชวนคนมาดูอีเวนต์ที่ตัวเองร่วมแสดง
แบบลำลอง (เพื่อนสนิท): V（て形（けい））＋くれない？ เช่น 見（み）に来（き）てくれない？
ตัวอย่าง:

友（とも）だちが行（い）けなくなってしまったので、落語（らくご）、いっしょに行（い）ってくれませんか？
日曜日（にちようび）の野球（やきゅう）の試合（しあい）、出（で）てくれない？






💬 คำศัพท์เสริม — ตอนที่ 1 (กีฬาที่ชอบ)

คำความหมาย気軽（きがる）にできるทำได้แบบสบายๆ ไม่ต้องคิดมาก運動（うんどう）が得意（とくい）じゃないไม่ถนัดออกกำลังกาย自分（じぶん）に合（あ）っているเหมาะกับตัวเองスピードと迫力（はくりょく）があるมีความเร็วและความอลังการペースจังหวะ, สปีด苦手（にがて）（な）ไม่ถนัด, ไม่ชอบ少（すく）ないน้อย勝負（しょうぶ）การแพ้ชนะ, การแข่งขัน景色（けしき）ทิวทัศน์興奮（こうふん）するตื่นเต้น迷惑（めいわく）をかけるสร้างความเดือดร้อนให้คนอื่น力士（りきし）นักซูโม่3分（ぶん）の1（いち）หนึ่งในสาม見（み）る専門（せんもん）ดูอย่างเดียว (ไม่เล่นเอง)場所（ばしょ）สถานที่ / ฤดูแข่งซูโม่興味（きょうみ）ความสนใจ嫌（いや）（な）ไม่อยากทำ嫌（きら）い（な）ไม่ชอบ競争（きょうそう）するแข่งขัน体育（たいいく）วิชาพละ連（つ）れて行（い）くพาไปด้วย

💬 คำศัพท์เสริม — ตอนที่ 2 (ชวนไปงานอีเวนต์)

คำความหมาย発表会（はっぴょうかい）งานแสดงผลงาน半年（はんとし）かけてใช้เวลาครึ่งปี演劇（えんげき）ละครเวที登場人物（とうじょうじんぶつ）ตัวละคร演（えん）じるแสดง (บทบาท)集（あつ）めるรวบรวมふれるได้สัมผัส/ได้ประสบการณ์機会（きかい）โอกาส映像（えいぞう）ภาพ/วิดีโอ日本的（にほんてき）（な）แบบญี่ปุ่นทั่วไปかける（時間（じかん）を）ใช้เวลาしかたないช่วยไม่ได้集合（しゅうごう）するรวมตัว, นัดพบ


🎭 ความรู้เสริมวัฒนธรรม

落語（らくご） — ราคุโกะ
ศิลปะการเล่าเรื่องตลกดั้งเดิมของญี่ปุ่น เริ่มในยุคเอโดะ ผู้เล่าคนเดียว (落語家（らくごか）) แสดงเป็นหลายตัวละครโดยใช้แค่พัด（扇子（せんす））กับผ้าเช็ดมือ（手（て）ぬぐい）เป็นอุปกรณ์ จบด้วยมุกตลก (オチ)

にほん vs にっぽん


にほん = ใช้ในชีวิตประจำวันทั่วไป, คำประสม เช่น 日本語（にほんご）, 日本料理（にほんりょうり）
にっぽん = อ่านแบบเก่ากว่า มักใช้ในบริบททางการ/ดั้งเดิม



📖 第2課（だい2か）ドラマを見（み）るのがいちばん好（す）きです

トピック: 好（す）きなもの好（す）きなこと (สิ่งที่ชอบ)

เป้าหมาย (B1): พูดคุยกับเพื่อนเรื่องภาพยนตร์ พร้อมแสดงความคิดเห็น/คอมเมนต์ได้ค่อนข้างละเอียด

โครงเรื่อง: ①試合（しあい）のシーンが本当（ほんとう）にドキドキしました (คุยความรู้สึกหลังดูหนัง/ละคร) + เนื้อหาเกี่ยวกับเกม, ซีรีส์, มังงะ ที่ชื่นชอบ


🔤 คันจิของบท（漢字（かんじ）のことば）

คันจิ＋ふりがなความหมาย (ไทย)共感（きょうかん）ความเห็นอกเห็นใจ, เข้าใจร่วมお腹（なか）ท้อง主人公（しゅじんこう）ตัวเอก (ของเรื่อง)歌詞（かし）เนื้อเพลง俳優（はいゆう）นักแสดง映像（えいぞう）ภาพ/วิดีโอ演（えん）じるแสดง (บทบาท)深（ふか）みความลึกซึ้ง探（さが）すค้นหา原作（げんさく）ต้นฉบับ (นิยาย/มังงะที่ดัดแปลง)死（し）ぬตาย


📝 ไวยากรณ์ (文法（ぶんぽう）ノート)

➊ S（普通形（ふつうけい））＋でしょう？


おもしろかったでしょう？ (สนุกใช่ไหมล่ะ)




ใช้ขอความเห็นด้วย/ยืนยันสิ่งที่ผู้พูดเชื่อว่าเป็นจริง โดยคาดว่าอีกฝ่ายจะรู้สึกเหมือนกัน
ออกเสียงลงท้ายแบบเสียงสูงขึ้น (rising intonation)
คำนาม/นะ-adjective → Nでしょう？ / ナA-でしょう？
ใช้กับคนสนิท (เพื่อน/ครอบครัว) เท่านั้น — ใช้กับผู้ใหญ่กว่า/เจ้านายจะดูเสียมารยาท
ตัวอย่าง:

この曲（きょく）、いい曲（きょく）でしょう？ → はい。だれの曲（きょく）ですか？
おすすめしてもらったドラマ、見（み）ました。→ どうだった？よかったでしょう？





➋ S（普通形（ふつうけい））＋気（き）がする


ストーリーがちょっと短（みじか）かった気（き）がしました。 (รู้สึกว่าเรื่องมันสั้นไปนิดนึง)




ใช้บอกความประทับใจ/ความเห็นแบบนุ่มนวล ไม่พูดตรงๆ
ในบทนี้ใช้เลี่ยงการพูดความเห็นเชิงลบแบบตรงๆ
นะ-adjective → ナA-な気（き）がする
ตัวอย่าง:

でも、最後（さいご）のシーンは、ちょっと、いまいちな気（き）がしました。
うーん、おもしろかったけど、ストーリーがちょっと難（むずか）しかった気（き）がする。





➌ S（普通形（ふつうけい））＋って N


歴史（れきし）を変（か）えようとする敵（てき）と戦（たたか）うってゲームです。 (เป็นเกมที่ต้องสู้กับศัตรูที่พยายามเปลี่ยนประวัติศาสตร์)




ใช้อธิบายเนื้อหาอย่างเจาะจง เช่น อธิบายเกม/ละคร
ต่อกับรูปธรรมดา (普通形（ふつうけい）) คำนาม/นะ-adjective → N1だってN2 / ナA-だってN
「って」= ภาษาพูดลำลอง ถ้าเป็นภาษาเขียน/ทางการใช้ という แทน
ตัวอย่าง:

最後（さいご）は、お金（かね）じゃなくて、人（ひと）とのつながりが大事（だいじ）だって話（はなし）です。
主人公（しゅじんこう）の男（おとこ）の子（こ）が、仲間（なかま）といっしょに次々（つぎつぎ）に事件（じけん）を解決（かいけつ）していくって映画（えいが）です。





➍ สำนวนเชื่อมประโยค（接続（せつぞく）の表現（ひょうげん））


それに、ゲームだけじゃなくて、舞台（ぶたい）とかミュージカルとかアニメとかもあって、いろいろ楽（たの）しめるんです。



ใช้เชื่อมประโยคเวลาต้องเรียงเหตุผล/จุดเด่นหลายๆ อย่าง ต่อกัน แบ่งตามความสัมพันธ์ของประโยค:

กลุ่มความหมายคำเชื่อมตัวอย่างบอกผล/สรุปจากเหตุผลก่อนหน้าだから (เพราะฉะนั้น)このレストラン、この前（まえ）『孤独（こどく）のグルメ』に出（で）てきたんですよ。だから、こんなに混（こ）んでいるんです。บอกผล (ภาษาพูด)それで／で (ดังนั้น)ライブのチケットがとれたんです。それで、その日（ひ）は休（やす）みをとりました。ขัดแย้งกับประโยคก่อนหน้าでも (แต่)趣味（しゅみ）はオンラインゲームです。でも、スポーツもよくします。เพิ่มเติมสิ่งอื่นあと／それと／それに (นอกจากนี้)この俳優（はいゆう）さん、歌（うた）も上手（じょうず）なんです。あと、ダンスもできるんですよ。


💬 คำศัพท์แสดงความรู้สึกหลังดูหนัง/ละคร（映画（えいが）の感想（かんそう））

😀 เชิงบวก

คำความหมายとてもよかったดีมาก迫力（はくりょく）があったสมจริง/อลังการ楽（たの）しかったสนุกすごいと思（おも）ったคิดว่าเจ๋งมาก感動的（かんどうてき）だったประทับใจ/ซาบซึ้งドキドキしたใจเต้นตุ๊บๆ ตื่นเต้น深（ふか）みがあったมีความลึกซึ้งきれいだったสวยงามすばらしかったยอดเยี่ยม興奮（こうふん）したตื่นเต้น

😞 เชิงลบ

คำความหมาย退屈（たいくつ）だったน่าเบื่อ難（むずか）しかったเข้าใจยาก物足（ものた）りなかったยังไม่พอ/ขาดอะไรบางอย่าง短（みじか）かったสั้นไปいまいちだったธรรมดา ไม่ประทับใจひどかったแย่มาก

💬 คำศัพท์เสริมอื่นๆ

คำความหมายクリアファイルแฟ้มใส結果（けっか）ผลลัพธ์全巻（ぜんかん）ครบทุกเล่ม (ของมังงะ)合（あ）わせるทำให้เข้ากัน, ผสมผสานリンクลิงก์, การเชื่อมโยงロック／ジャズ／クラシックร็อก/แจ๊ส/คลาสสิกハマるติดใจ, หลงใหล印象的（いんしょうてき）（な）น่าประทับใจそっくりเหมือนมาก


🎭 ความรู้เสริมวัฒนธรรม

マンガアプリ — แอปอ่านมังงะ
คนญี่ปุ่น (โดยเฉพาะวัยรุ่น) นิยมอ่านมังงะผ่านสมาร์ตโฟนมากกว่าหนังสือกระดาษ มีทั้งแบบฟรีทั้งหมด ฟรีบางตอน ดูโฆษณาแลกฟรี หรือ "รอแล้วฟรี" (待（ま）てば無料（むりょう）)

オノマトペ — คำเลียนเสียง/สภาพ
ภาษาญี่ปุ่นใช้คำอุทานเลียนเสียง (擬声語（ぎせいご）/擬音語（ぎおんご）) เช่น ワンワン (เห่า), คำเลียนอาการ (擬態語（ぎたいご）) เช่น ピカピカ (แวววาว), และคำแสดงอารมณ์ (擬情語（ぎじょうご）) เช่น イライラ (หงุดหงิด), わくわく (ตื่นเต้นดีใจ)

ふりがなの特別（とくべつ）な用法（ようほう）
บางครั้งคันจิในมังงะ/เนื้อเพลงจะมีฟุริงานะที่ไม่ใช่การอ่านปกติ เช่น 地球（ほし）อ่านว่า「ほし」แทน「ちきゅう」เพื่อสื่อความหมายพิเศษ

スラムダンク — สแลมดังก์
