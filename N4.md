## ส่วนที่ 1: ไวยากรณ์ (Grammar)
 
### 1.1 รูปกริยา (Verb Forms) พื้นฐานที่ต้องแม่น
 
| รูป | ตัวอย่าง (食べる) | การใช้ |
|---|---|---|
| て形 | 食べて | ขอร้อง เชื่อมประโยค กำลังทำ |
| た形 | 食べた | อดีต |
| ない形 | 食べない | ปฏิเสธ |
| 辞書形 | 食べる | รูปพจนานุกรม |
| 意向形 ( willing) | 食べよう | ชวนทำ/ตั้งใจทำ |
| 可能形 (potential) | 食べられる | ความสามารถ |
| 受身形 (passive) | 食べられる | ถูกกระทำ |
| 使役形 (causative) | 食べさせる | ให้/บังคับทำ |
| 命令形 | 食べろ | คำสั่ง (หยาบ) |
| 条件形 (ば) | 食べれば | เงื่อนไข |
 
### 1.2 ไวยากรณ์แสดงความสามารถ / อนุญาต / ห้าม
 
- **〜ことができる** = สามารถทำได้ (泳ぐことができる)
- **可能形 (られる/える)** = สามารถ (食べられる、書ける)
- **〜てもいいです** = ทำก็ได้ (ขออนุญาต)
- **〜てはいけません** = ห้ามทำ
- **〜なければなりません / 〜なくてはいけません** = ต้องทำ
- **〜なくてもいいです** = ไม่ทำก็ได้
### 1.3 ไวยากรณ์แสดงเงื่อนไข (Conditional)
 
- **〜と** = ถ้า...แล้วจะ (ผลลัพธ์แน่นอน/อัตโนมัติ) — ボタンを押すとドアが開きます
- **〜ば** = ถ้า (เงื่อนไขทั่วไป) — 安ければ買います
- **〜たら** = ถ้า/พอ...ก็ (ใช้บ่อยที่สุด) — 雨が降ったら行きません
- **〜なら** = ถ้าเป็นเรื่อง...ล่ะก็ — 行くなら早く行って
### 1.4 ไวยากรณ์แสดงการให้-รับ (やり・もらい)
 
- **あげる** = ให้ (ผู้พูด→คนอื่น หรือคนอื่น→คนอื่น)
- **もらう** = ได้รับ
- **くれる** = ให้ (คนอื่น→ผู้พูด)
- **〜てあげる / 〜てもらう / 〜てくれる** = ทำให้ (ในรูปกริยา)
### 1.5 ไวยากรณ์การกระทำที่กำลังดำเนิน / ผลลัพธ์
 
- **〜ている** = กำลังทำ / สภาพที่เป็นผล (ドアが開いている)
- **〜てある** = สภาพที่ถูกจัดเตรียมไว้ (準備してある)
- **〜ておく** = ทำเตรียมไว้ล่วงหน้า
- **〜てしまう** = ทำเสร็จสิ้น/เสียดาย/ทำพลาด
### 1.6 ไวยากรณ์บอกความตั้งใจ / แผนการ
 
- **〜つもりです** = ตั้งใจว่าจะ
- **〜(よ)うと思います** = คิดว่าจะทำ
- **〜予定です** = มีกำหนดการว่าจะ
- **〜ことにする** = ตัดสินใจว่าจะทำ
- **〜ことになる** = มีการตัดสินใจ/กลายเป็นว่า (มักไม่ใช่ผู้พูดตัดสิน)
### 1.7 ไวยากรณ์เปรียบเทียบ
 
- **AはBより〜** = A ...กว่า B
- **AとBとどちらが〜** = A กับ B อันไหน...กว่ากัน
- **〜の中で一番〜** = ในบรรดา... ...ที่สุด
### 1.8 ไวยากรณ์บอกเหตุผล
 
- **〜から** = เพราะว่า (เน้นเหตุผลส่วนตัว)
- **〜ので** = เพราะว่า (สุภาพ/เป็นทางการกว่า)
- **〜ため(に)** = เพื่อ/เนื่องจาก
### 1.9 ไวยากรณ์บอกจุดประสงค์
 
- **〜ために** = เพื่อที่จะ
- **〜に行く/来る** = ไปทำ/มาทำ (V-stem+に行く, N+に行く)
### 1.10 ไวยากรณ์การคาดเดา / ได้ยินมา
 
- **〜そうです (伝聞)** = ได้ยินว่า (普通形+そうです)
- **〜そうです (様態)** = ดูเหมือนจะ (V-stem/い-stem+そうです)
- **〜ようです** = ดูเหมือนว่า (จากการสังเกต)
- **〜みたいです** = ดูเหมือนว่า (แบบพูด casual)
- **〜でしょう** = คงจะ/น่าจะ
- **〜かもしれません** = อาจจะ
### 1.11 ไวยากรณ์คำสั่ง / ขอร้อง / ห้าม
 
- **〜てください** = กรุณาทำ
- **〜ないでください** = กรุณาอย่าทำ
- **〜なさい** = ทำซะ (คำสั่งสุภาพปานกลาง เช่นพ่อแม่พูดกับลูก)
### 1.12 ไวยากรณ์อื่นๆ ที่ออกสอบบ่อย
 
- **〜すぎる** = มากเกินไป (食べすぎる)
- **〜やすい / 〜にくい** = ง่าย/ยากที่จะทำ
- **〜方 (かた)** = วิธีทำ (使い方)
- **〜始める / 〜終わる / 〜続ける** = เริ่ม/จบ/ทำต่อเนื่อง
- **〜前に / 〜後で** = ก่อน/หลัง
- **〜間 / 〜間に** = ระหว่าง (ตลอด)/ในระหว่าง (ช่วงหนึ่ง)
- **〜ながら** = ทำ...ไปพร้อมกับ
- **〜たり〜たりする** = ทำโน่นทำนี่ (ยกตัวอย่างกิจกรรม)
- **〜し〜し** = ทั้ง...ทั้ง... (ให้เหตุผลหลายข้อ)
- **〜ばかり** = เพิ่ง.../มีแต่...
- **〜のに** = ทั้งๆ ที่ (แสดงความขัดแย้ง/ผิดคาด)
- **〜けど / 〜が** = แต่
- **敬語เบื้องต้น**: 尊敬語 (いらっしゃる、召し上がる) และ 謙譲語 (まいる、いただく) แบบพื้นฐาน
- **〜という** = ที่เรียกว่า
- **〜はずです** = น่าจะ (มั่นใจตามเหตุผล)
- **〜がる** = แสดงท่าทีว่ารู้สึก (บุคคลที่ 3) เช่น 欲しがる

## เคล็ดลับการทบทวน
 
1. **ไวยากรณ์**: จำรูปประโยคพร้อมตัวอย่างจริง อย่าท่องแค่สูตร ลองแต่งประโยคของตัวเอง 2-3 ประโยคต่อไวยากรณ์
2. **คำศัพท์**: จัดกลุ่มตามหมวดหมู่และคันจิคู่กัน (การอ่าน+ความหมาย+ตัวอย่างประโยค)
3. **ฝึกข้อสอบเก่า**: N4 เน้นไวยากรณ์ผสมและการอ่านย่อหน้าสั้น ควรฝึกอ่าน 読解 บ่อยๆ
4. **ฟัง**: ฝึก 聴解 ด้วยบทสนทนาสั้นๆ ในชีวิตประจำวัน
---
 
*เอกสารนี้ครอบคลุมไวยากรณ์หลักและคำศัพท์ตัวอย่างของ N4 หากต้องการแบบฝึกหัด, แบบทดสอบ, หรือคำศัพท์เพิ่มเติมเฉพาะหมวด (เช่น กริยากลุ่ม 3, คำเชื่อมทั้งหมด) แจ้งได้เลยครับ
